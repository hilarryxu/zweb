zweb
====

A web framework for tornado with orm.


Installing
----------

Install and update using `pip`_:

.. code-block:: text

    $ pip install git+https://github.com/hilarryxu/zweb.git#egg=zweb

.. _pip: https://pip.pypa.io/en/stable/quickstart/
